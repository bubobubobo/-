## 요구사항

### 메인 페이지

1. "Sign in" 버튼을 클릭하면 로그인 페이지가 팝업된다.
2. "Sign up" 버튼을 클릭하면 회원가입 페이지가 팝업된다.
3. "Hit me!" 버튼을 클릭하면 리셋된 스톱워치와 함께 질문 창과 오른쪽 물음표 버튼이 나타난다.
4. 물음표 버튼을 클릭하면 예시답안이 토글된다.
